Meta-prompt

Ayúdame a migrar este código de Java 8 a Java 21 o de jQuery a React.

Prompt

"Actúa como un **Experto en Migraciones de Software**. Tengo un componente antiguo escrito en **jQuery** y quiero migrarlo a un componente funcional de **React (con Hooks)**. Analiza la lógica y propón la nueva implementación."

Código Legacy (jQuery):

$('.btn-toggle').click(function() {
  var id = $(this).data('id');
  $('.content-' + id).slideToggle();
  $(this).toggleClass('active');
});

Entregables

  • Componente React: Código completo usando `useState`.
  • Explicación: ¿Cómo se mapean los eventos de jQuery al Virtual DOM de React?
  • Testing: Un test unitario básico con React Testing Library.

Ejemplo de Output Estructurado

El código generado:

const ToggleComponent = ({ id, content }) => {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <div>
      <button 
        className={`btn-toggle ${isOpen ? 'active' : ''}`}
        onClick={() => setIsOpen(!isOpen)}
      >
        Toggle
      </button>
      {isOpen && <div className={`content-${id}`}>{content}</div>}
    </div>
  );
};
