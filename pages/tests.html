Generación de Tests

Meta-prompt

dame un prompt para generar una suite de tests completa para esta clase

Prompt

"Actúa como un **QA Automation Engineer**. Escribe una suite de pruebas unitarias exhaustiva para el siguiente código usando [Framework, ej. Jest, Pytest, JUnit]."

Asegúrate de cubrir los siguientes escenarios:

  • Happy Path: El flujo normal donde todo funciona correctamente.
  • Edge Cases: Entradas vacías, nulas, límites de arrays, números negativos, etc.
  • Error Handling: Verifica que las excepciones se lancen y capturen como se espera.
  • Mocking: Simula todas las dependencias externas (bases de datos, APIs) para que los tests sean aislados y rápidos.

Plan de Pruebas (Test Plan)

Estructura los casos de prueba propuestos:

Caso de Prueba Input Resultado Esperado
Calcula total correctamente Items: [{price: 10}, {price: 20}] Total: 30
Maneja lista vacía Items: [] Total: 0 (No error)
Lanza error con precio negativo Items: [{price: -5}] Exception: "Invalid Price"
