Onboarding Acelerado

Meta-prompt

dame un prompt para hacer onboarding en este proyecto y entender su arquitectura rápidamente

Prompt

"Actúa como un **Tech Lead** empático y didáctico. Tu objetivo es ayudar a un nuevo desarrollador a entender la arquitectura del proyecto y a configurar su entorno local en menos de 15 minutos."

Genera un documento que cubra:

  • Big Picture: Diagrama de alto nivel de los módulos y cómo se comunican.
  • Tech Stack: Lista de tecnologías clave (Frontend, Backend, DB) y por qué se eligieron.
  • Directorios Clave: Dónde encontrar la lógica de negocio, las vistas, los tests, etc.
  • Flujo de Trabajo: Cómo correr el servidor localmente, cómo ejecutar los tests y cómo hacer un deploy.

Guía de Inicio Rápido (Quickstart)

Paso Comando Descripción
1. Instalar dependencias `npm install` Descarga las librerías necesarias.
2. Configurar entorno `cp .env.example .env` Crea las variables de entorno locales.
3. Iniciar servidor `npm run dev` Levanta la app en `http://localhost:3000`.

Prompts Adicionales

  • "Glosario del Dominio": "Crea un diccionario de términos específicos del negocio que se usan en este código (Ubiquitous Language)."
  • "Mapa de Dependencias": "Lista las dependencias externas más críticas y explica qué rol cumple cada una."
