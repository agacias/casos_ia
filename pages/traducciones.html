Traducciones e i18n

Meta-prompt

dame un prompt para hacer que mi código soporte internalización en inglés y español

Prompt

"Actúa como un experto en **L10n (Localización) e i18n (Internacionalización)**. Analiza el código fuente y refactorízalo para eliminar todos los textos 'hardcoded'."

Realiza las siguientes tareas:

  • Extracción de Strings: Mueve todos los textos visibles por el usuario a archivos JSON de recursos separados (ej. `es.json`, `en.json`).
  • Claves Semánticas: Usa claves descriptivas y anidadas (ej. `home.hero.title`) en lugar de claves ambiguas.
  • Interpolación: Asegúrate de que las variables (nombres, fechas) se manejen con placeholders (ej. `Hola, {name}`).
  • Pluralización: Gestiona correctamente las reglas de plurales para cada idioma.

Estrategia de Recursos

Clave (Key) Español (es.json) Inglés (en.json)
`buttons.submit` "Enviar Formulario" "Submit Form"
`messages.welcome` "Bienvenido, {name}" "Welcome, {name}"
`errors.required` "Este campo es obligatorio" "This field is required"

Prompts Adicionales

  • "Traducción de Contenido": "Toma este archivo JSON en español y genera la versión en inglés (en.json) manteniendo el tono formal."
  • "Validación de i18n": "Escanea los componentes de React y dime si me he dejado algún texto sin envolver en la función `t()`."
