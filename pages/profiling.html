Prompt

"Actúa como un **Principal Software Engineer** experto en **Performance Optimization**. Realiza un análisis de rendimiento estático del siguiente código."

Enfócate en identificar:

  • Cuellos de Botella (Bottlenecks): Algoritmos con complejidad O(n^2) o superior innecesaria.
  • Problemas de Memoria: Fugas de memoria, creación excesiva de objetos o referencias circulares.
  • Consultas Ineficientes: Problemas de N+1 en bases de datos o llamadas bloqueantes a APIs.
  • Bloqueos de Hilo Principal: Operaciones intensivas que congelan la UI (en JS) o el Event Loop.
