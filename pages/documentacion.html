Documentación Automática

Meta-prompt

dame un prompt para generar la documentación técnica y un diagrama para este componente

Prompt

"Actúa como un **Technical Writer** con experiencia en Developer Experience (DX). Analiza el código proporcionado y genera la documentación necesaria para que cualquier desarrollador pueda entenderlo y usarlo."

El entregable debe incluir:

  • Descripción General: ¿Qué hace este módulo y por qué existe?
  • API Reference: Tabla de props, métodos, parámetros y tipos de retorno.
  • Ejemplos de Uso: Snippets de código mostrando los casos de uso más comunes.
  • Diagramas: Usa sintaxis Mermaid.js para visualizar flujos.

Estructura de Documentación Sugerida

Sección Contenido Formato
Header Título, Badges (versión, build status), Descripción corta. Markdown
Installation `npm install @package/name` Code Block
Props Table Argumentos, Tipos, Default Values, Descripción. Table (Markdown)

Ejemplo de Diagrama (Mermaid)

```mermaid graph TD; A[Usuario] -->|Login| B(Auth Service); B --> C{Es válido?}; C -->|Si| D[Generar Token]; C -->|No| E[Error 401]; ```

Prompts Adicionales

  • "Docstrings": "Añade comentarios en formato JSDoc / Docstring a todas las funciones públicas, explicando parámetros y valores de retorno."
  • "De Código a Texto": "Explica este algoritmo complejo en lenguaje natural (Plain English) para una audiencia no técnica."
