DevOps & CI/CD

Meta-prompt

Crea un Dockerfile optimizado y un pipeline de GitHub Actions.

Prompt

"Actúa como un **DevOps Engineer**. Necesito containerizar una aplicación Node.js y automatizar su despliegue. Genera los siguientes archivos:"

Dockerfile

Debe ser multi-stage para optimizar el tamaño de la imagen final.

# Build Stage
FROM node:18-alpine AS builder
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build

# Production Stage
FROM node:18-alpine
WORKDIR /app
COPY --from=builder /app/dist ./dist
COPY --from=builder /app/node_modules ./node_modules
CMD ["node", "dist/main.js"]

GitHub Actions (ci.yaml)

  • Trigger: Push a la rama main o Pull Requests.
  • Jobs:
    • Run Tests: `npm test`
    • Build Docker Image: Y subirla a Docker Hub (usa secretos).
